## :fire: How to run

To start the **project** clone this repository:
```bash 
$ git clone https://github.com/jesspsilva/recipes-tailwind
```
Then use the commands bellow:
```bash
cd recipes-tailwind
npm install
```
After this process open the index.html file (public/index.html).
